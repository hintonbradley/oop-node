Constructors

A constructor is a special method for creating and initializing an object created within a class..

Constructor Object

function Car(make, model) {
  // Public members
  this.make = make;
  this.model = model;
  this.accelerate = function() {
    console.log('This ' + this.model + ' is accelerating to 60mph.');
  }
  // Private member
  const driver = 'Michael Knight';
};
There are constructors for all Javascript data types
Strings:
  let myString = 'This is a string';
  let myConstructorString = new String ('This is also a string');

  myString.constructor

Numbers:

Functions:
  const myFunction = function (value) {
    console.log(value);
  };

  const myFunction = new Function ('value', `console.log(value);`);

  myFunction.constructor
    => ƒ Function() { [native code] }

  myConstructorFunction.constructor
    => ƒ Function() { [native code] }
Because of these built in JS constructors, there are many functions and properties available for each data type.
Example, for functions, we have the 'call' and 'apply' methods:
let burgerKing = {
  food: 'Whopper',
  placeOrder: function () {
    console.log("I'll have a " + this.food)
  }
};
  orderDrink: function (liquid) {
    console.log("I'll have a " + liquid + ' with my ' + this.food)
  }
};
let mcDonalds = {
  food: 'Big Mac',
  orderFood: burgerKing.placeOrder
};

// if adding parameters, can add them after the 'this' reference

burgerKing.orderDrink.call(mcDonalds, 'Coke')

=> "I'll have a Coke with my Big Mac"

// using an array to add parameters

burgerKing.orderDrink.apply(mcDonalds, ['Coke'])

=> "I'll have a Coke with my Big Mac"
