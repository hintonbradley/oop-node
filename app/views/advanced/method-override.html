Overriding Prototype Methods

If we have many objects inheriteing from a super class or classes, we can create an 'extends' function to reduce our code and simplify the inheritence process. (You should be able to call the Parallelogram's prototype method on both the rectangle and square instances in the console.)

Before we can override a method on a prototype we need to first change the prototype of a child constructor to inherit properties and methods of a super constructor.

Only after we reset the prototype of a child class can we then override a method that's already on the new super prototype.
