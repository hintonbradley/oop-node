Calling Super Constructor

(open console to view)

In order for a child constructor to gain access to it's super's own properties and methods, you must call the super constructor.

Parallelogram

the Square's Super constructor

function Parallelogram(color) {
  this.color = color;
  this.fourSides = true;
  this.oppositeEqualSides = true;
  this.oppositeAnglesEqual = true;
};

Parallelogram.prototype.countSides = function () {
  return 4;
}

Square

a child of the Parallelogram constructor

function Square() {
  this.allSidesEqual = true;
  this.angleDegrees = 90;
};

// In order for the square instances to inherit the parallelogram's own properties, we will need to call the Parallelogram constructor inside the Square constructor. We do this using the 'call' method to set 'this' to the already created Square object.

function Square() {
  this.allSidesEqual = true;
  this.angleDegrees = 90;
  Parallelogram.call(this, 'yellow')
};

Note: if we were to use the 'new' keyword to call the Parallelogram constructor inside the square object, Javascript would have created a whole new object instead of using the square instance object that was created when we use the 'new' keyword on the Square object.
