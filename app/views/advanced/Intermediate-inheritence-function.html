Intermediate inheritence function

If we have many objects inheriteing from a super class or classes, we can create an 'extends' function to reduce our code and simplify the inheritence process. (You should be able to call the Parallelogram's prototype method on both the rectangle and square instances in the console.)

In the below example we have two child classes (Square and Rectangle) inheriting from a super class (Parallelogram). Our inheritence code is repeated twice in the code below.

...
  // Writing code to have Square inherit Parallelogram's members
    Square.prototype = Object.create(Parallelogram.prototype);
    Square.prototype.constructor = Square;

  // Duplicating code to have Rectangle inherit Parallelogram's members
    Rectangle.prototype = Object.create(Parallelogram.prototype);
    Rectangle.prototype.constructor = Rectangle;
...
