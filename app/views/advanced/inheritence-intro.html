Inheritence

Inheritance is the concept of one thing gaining the properties or behaviours of something else. You must use a special object called prototype which assigns properties to a type (such as Parallelogram). Only functions have a prototype, which is why you must create a function first.

Parallelogram

the Square's Super constructor

function Parallelogram() {
  this.fourSides = true;
  this.oppositeEqualSides = true;
  this.oppositeAnglesEqual = true;
};

Parallelogram.prototype.countSides = function () {
  return 4;
}

Square

a child of the Parallelogram constructor

function Square() {
  this.allSidesEqual = true;
  this.angleDegrees = 90;
};

//In order for the Square instances to inherit the Parallelogram prototype's properties and methods we do the following:
Square.prototype = Object.create(Parallelogram.prototype);

// NOTE! Whenver you change an object's prototype, you must also change the constructor back to the original constructor.
Square.prototype.constructor = Square;

//This will allow any new Square instance to inherit the Parallelogram's prototype methods and properties, but not the Parallelogram's instance properties/methods.
let spongeBob = new Square();

spongeBob.countSides();
  => 4   //because countSides is a prototype method
spongeBob.fourSides;
  => undefined   //because fourSides is an instance property
