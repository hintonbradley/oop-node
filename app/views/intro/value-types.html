Reference Types
  • Objects
  • Arrays
  • Functions
let myObj = {value: 10}

function increase (myObj) {
  myObj.value++;
}

increase(myObj);

console.log(myObj.value)
=> 11

In this case the original object's value will increase because you are passing the object into the function and therefore passing the reference to the object. This would not be the case with primitive value types (see below).
