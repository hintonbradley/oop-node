Abstraction

Hiding the details and showing only the essentials in order to reduce complexity and increase efficiency.

(Open console to manipulate and view results)

Private variables: Variables and methods the public can not interface with nor manipulate.

e.g. You don't want a user to open the elevator doors while it's in motion.

Public variables: Variables and methods that the public can interface with.

e.g. The user is only able to select a floor number. Everything else happens behind the scenes.
