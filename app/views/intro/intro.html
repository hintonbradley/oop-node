Object Literal Notation

const circle1 = {
  //Properties - key value pairs inside objects
  radius: 1,
  location: {
    x: 1,
    y: 1
  },

  //Methods - functions inside objects
  show: function () {
    console.log('Showing an Object Literal Notation circle');
  }
};
To loop through an object's key value pairs:
  for (let key in circle1) {
    console.log(key, circle1[key]);
  }
  => radius 1
        location {x: 1, y: 1}
        show Æ’ () { console.log('Showing an Object Literal Notation circle'); }
To collect all the keys of an object:
  const keys = Object.keys(circle1);
  console.log(keys);
    => ["radius","location","show"']
To find if a property is in an object:
  if ('radius' in circle1) {
    console.log('Circle has a radius');
  }
