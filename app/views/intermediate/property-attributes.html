Property Attributes

All properties of objects have attributes attached to them.

let myCar = {
   make: "Porche",
  model: "911"
};

console.log(Object.getOwnPropertyDescriptor(myCar, 'make'));
  =>
    configurable: true
    enumerable: true
    value: "Porche"
    writable: true

Even prototype properties have these attributes:


  let objectBase = Object.getPrototypeOf(myCar);
  let baseDescript = Object.getOwnPropertyDescriptor(objectBase, 'toString');
  console.log(baseDescript);

configurable
Ability to delete the property.
Defaults to false.
enumerable
Basically, we can visibly loop over them.
Defaults to false.
value
The value associated with the property. Can be any valid JavaScript value (number, object, function, etc).
Defaults to undefined.
writable
Ability to change the value associated with the property may be changed.
Defaults to false.
