Property Attributes

All properties of objects have attributes attached to them.
let myCar = {
   make: "Porche",
  model: "911"
};

console.log(Object.getOwnPropertyDescriptor(myCar, 'make'));
  =>
    configurable: true
    enumerable: true
    value: "Porche"
    writable: true

Even prototype properties have these attributes:


  let objectBase = Object.getPrototypeOf(myCar);
  let baseDescript = Object.getOwnPropertyDescriptor(objectBase, 'toString');
  console.log(baseDescript);

    configurable: true
    enumerable: false

    writable: true

configurable
Ability to delete the property.
enumerable
Basically, we can visibly loop over them.
value
The value associated with the property. Can be any valid JavaScript value (number, object, function, etc).
writable
Ability to change the value associated with the property may be changed.
let muppets = {
  frog: "Kermit",
  bear: "Fozzie",
  dog: "Ralph"
};
//To prevent the ability to change the value of a variable:
Object.defineProperty(muppets, "frog", {
  writable: false
})
muppets.frog = 'scooter';
console.log(muppets.frog)
  => Kermit
//To prevent the ability to enumerate over a key-value pair:
Object.defineProperty(muppets, "bear", {
  enumerable: false
})
console.log(Object.keys(muppets));
  => ["frog", "dog"]
//To prevent the ability to delete a key-value pair:
Object.defineProperty(muppets, "dog", {
  configurable: false
})
delete muppets.dog;
console.log(muppets)
  => {frog: "Kermit", dog: "Ralph", bear: "Fozzie"}
