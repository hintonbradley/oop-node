Overwriting methods using Prototype

Open console to view.

Dog constructor

const Dog = function (name) {
  this.name = name;
  this.food = 'Alpo';
  this.eat = function() {
    console.log(this.name + "is eating " + this.food);
  }
};


Before

//before changing Dog.prototype.toString method, rover.toString() is:
=>[object Object]

Dog.prototype.toString = function() { return "This built in Javascript method has been altered" };


After

//After changing Dog.prototype.toString method, rover.toString() is:
=> "This built in Javascript method has been altered"


The prototype method will be used instead of the built-in Javascript method (on the base object) because the prototype is more accessible than the base object.
