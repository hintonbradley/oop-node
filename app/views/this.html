'this' keyword

The JavaScript 'this' keyword refers to the object it belongs to.
In the example below, the first 'this' refers to the john object because that object is the parent (owner) object.
However the second 'this' refers to the sister object since that is the parent (owner) object.

let john = {
  name: 'John Peterson',
  age: 29,
  logThis: function () {
    console.log('Inside the john object, "this" refers to: ', this)
  },
  sister: {
    name: 'Sue Peterson',
    age: 26,
    logThis: function () {
      console.log('Inside the john object, "this" refers to: ', this)
    }
  }
};

john.logThis();
=> john {...}

john.sister.logThis();
=> sister {...}

In an object method, this refers to the "owner" of the method.
In the Burger King example below, 'this' refers to the burgerKing object.
The burgerKing object is the owner of the placeOrder method.

However, when the same method is used in the McDonalds objct, 'this' reference changes to the mcDonalds object

let burgerKing = {
  food: 'Whopper',
  placeOrder: function () {
    console.log("I'll have a " + this.food)
  }
};

burgerKing.placeOrder();

=> "I'll have a Whopper"

let mcDonalds = {
  food: 'Big Mac',
  orderFood: burgerKing.placeOrder
  }
};

mcDonalds.orderFood();

=> "I'll have a Big Mac"
