Object Types

The simplest way to create an object is using an object literal notation

Object Literal

const circle1 = {
  radius: 1,
  location: {
    x: 1,
    y: 1
  },
  show: function () {
    console.log('Showing an Object Literal Notation circle');
  }
};

circle1.show();
  => "Showing an Object Literal Notation circle"

To create multiple objects with the same structure and behavior (methods), use a factory or a constructor.

Factory Object

function createCircle (radius) {
  return {
    radius,
    draw: function() {
      console.log('Drawing a Factory instansiated circle');
    }
  }
};

const circle2 = createCircle(1);
circle2.draw();
  => "Drawing a Factory instantiated circle"

When an object is created using a factory, the following things happen:

  • The built in Javascript contructor function is used to create the object
  circle2.constructor
  (built in JS object constructor)
  => ƒ Object() { [native code] }

Constructor Object

function Circle(radius) {
  console.log('Inside any instance of a constructor, "this" is: ', this);
  this.radius = radius;
  this.create = function() {
    console.log('Creating an instance of a circle from a constructor.');
  }
}

const circle3 = new Circle(1);
circle3.create();
  => "Creating an instance of a circle from a constructor."

When the 'new' keyword is used to create an object from a constructor, the following things happen:

  • An empty object is created
  • The 'this' keyword is bound to this new object
  • The new object is returned
  circle3.constructor
  (the above Circle constructor)
  => ƒ Circle(radius) {...}
